# クイックコピー＆分割

⌘⇧Cを使用して、「分割コピー」を行うことで、複数行を直接複数の項目に分割してコピーをペーストキューに保存できます。「貼り付け後にEnterを押す」オプションを選択すると、次のセルに自動的に切り替えて素早くフォーム入力ができます！
