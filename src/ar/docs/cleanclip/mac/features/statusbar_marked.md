# وظائف شريط القوائم

1. عرض آخر سجل نسخ
2. النقر الأيسر لفتح النافذة الرئيسية، النقر الأيمن لفتح القائمة
3. حجب سريع لمحتوى النسخ من التطبيق الحالي
4. إدارة الرقم التسلسلي
5. دليل البدء السريع
