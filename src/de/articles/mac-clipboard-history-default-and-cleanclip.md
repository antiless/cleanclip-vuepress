# Zwei Methoden zur Anzeige des Mac-Zwischenablageverlaufs: Tipps zur Effizienzsteigerung

Schnelle Tipps:

* Greifen Sie schnell auf die Zwischenablage zu, indem Sie Finder > Bearbeiten > Zwischenablage anzeigen öffnen.
* Verwenden Sie Anwendungen wie CleanClip, um das Zwischenablage-Erlebnis zu verbessern. Sie bieten umfangreiche Historien und zusätzliche Funktionen.

## Methode 1: Standard-Zwischenablage

**Schritt 1:** Öffnen Sie den **Finder** auf Ihrem Mac, wählen Sie in der Menüleiste **Bearbeiten** und klicken Sie dann auf **Zwischenablage anzeigen**.

![Zwischenablage anzeigen](/images/blogs/view_macos_clipboard_step1.png)

**Schritt 2:** Überprüfen Sie den zuletzt kopierten Text.

Wir haben versucht, über das Menü der Systemeinstellungen eine Tastenkombination für die Mac-Zwischenablage festzulegen, hatten jedoch keinen Erfolg. Die Standard-Zwischenablage von macOS ist nicht intuitiv und bietet viel Raum für Verbesserungen.

Lassen Sie uns einen Drittanbieter-Zwischenablage-Manager ansehen: CleanClip.

## Methode 2: Verwendet eine Drittanbieter-Anwendung

### CleanClip

CleanClip ist ein leistungsstarker Zwischenablage-Manager, den Sie problemlos im Mac App Store herunterladen können und der die Apple M-Serie Chips unterstützt. Lassen Sie uns ansehen, wie Sie CleanClip verwenden können.

**Schritt 1:** Öffnen Sie den **App Store** auf Ihrem Mac. Geben Sie im Suchfeld _CleanClip_ ein und **laden** Sie die Anwendung herunter.

![CleanClip im App Store suchen](/images/appstore_cleanclip.png)

**Schritt 2:** Öffnen Sie die **CleanClip**-Anwendung, aktivieren Sie **Beim Anmelden starten** und schon können Sie loslegen. Um die Anwendung anzupassen, fahren Sie mit den folgenden Schritten fort.

**Schritt 3:** Starten Sie **CleanClip**, klicken Sie auf das **CleanClip-Symbol** in der Menüleiste und öffnen Sie die **Einstellungen**.

![CleanClip-Einstellungen öffnen](/images/blogs/theme.png)

**Schritt 4:** Im Menü Allgemein können Sie Ton, Farbthema usw. einstellen und **Bereitstellen CleanClip beim Systemstart** aktivieren.

**Schritt 5:** Gehen Sie zum Menü **Ignorieren**, wo Sie CleanClip anweisen können, bestimmte Anwendungen nicht zu protokollieren.

![CleanClip Ausnahmeeinstellungen](/images/blogs/preferences-ignore.png)

Wann immer Sie Text oder Weblinks kopieren, läuft CleanClip im Hintergrund und speichert den Inhalt. Klicken Sie einfach auf das CleanClip-Symbol in der Menüleiste, um die vollständige Zwischenablage-Historie auf Ihrem Mac anzuzeigen. Sie können CleanClip jederzeit über die Menüleiste öffnen und auswählen, um den Verlauf zu löschen.

Die Hauptfunktionen von CleanClip umfassen:

1. Unendliche Zwischenablage-Historie
2. Intelligente Kategorisierung
3. Schnelle Suche
4. Einfügewarteschlange
5. Datenschutz

CleanClip bietet eine 14-tägige kostenlose Testversion, nach der Sie sich für ein Abonnement oder den einmaligen Kauf einer dauerhaften Lizenz entscheiden können. Die Preise betragen 1,99 USD pro Monat oder 19,99 USD für eine dauerhafte Lizenz.

Wenn Sie einen leistungsstarken und benutzerfreundlichen Zwischenablage-Manager suchen, ist CleanClip eine Überlegung wert. Es kann Ihre Produktivität erheblich steigern und das Kopieren und Einfügen einfacher und intelligenter machen.
