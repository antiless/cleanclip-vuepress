# 통일된 미리 설정된 붙여넣기 형식

미리 설정된 형식을 사용하면 모든 항목이 지정된 형식으로 붙여넣기됩니다.

현재 지원되는 형식은 순수 텍스트 붙여넣기, 전부 소문자로, 전부 대문자로 등이 있습니다.
