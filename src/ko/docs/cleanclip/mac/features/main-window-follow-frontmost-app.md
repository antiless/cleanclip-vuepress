# 프론트 앱 자동 추적

프론트 앱 추적 모드를 선택하면 앱 필터가 자동으로 프론트 앱의 변경사항을 추적하여 해당 앱의 이력 복사 기록을 표시합니다.

이 모드는 목록과 함께 사용하여 많은 가능성을 제공하며 각 앱에 대한 전용 수집 공간을 만들어 앱의 잠재력을 최대화할 수 있습니다.
