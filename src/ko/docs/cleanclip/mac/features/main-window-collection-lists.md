# 즐겨찾기 목록 - 수동 관리

항목을 하나 이상 선택한 후 마우스 오른쪽 버튼을 클릭하여 "목록에 추가"를 선택하면 해당 항목이 지정된 목록에 즐겨찾기됩니다.

즐겨찾기 목록은 스마트 목록으로 필터링 될 수 있어 더 많은 게임 방식을 구현할 수 있습니다.
