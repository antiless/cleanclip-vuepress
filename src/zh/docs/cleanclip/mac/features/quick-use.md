# 快速使用

1. ⌘；调起快捷菜单
2. a-z 直接开始搜索，输入 image 可搜索图片，输入 file 可搜索文件
3. 空格键预览
4. ⇥/⇧⇥ 上下翻页
5. 数字键/回车 粘贴对应的项
