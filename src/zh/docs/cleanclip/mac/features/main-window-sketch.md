# 收藏列表 - 自动收集

选择一个收藏列表，启用下方自动收集选项，你复制的内容会自动收集到指定目录。

你可以为它设置别名，以后随时打开这个目录，方便取用。

很多 App 特别是内容创作 App 缺少好用的收藏功能，比如 Sketch、Final Cut Pro 等等。

如果你的常用 App 不支持收藏夹功能，那么这个功能对你非常有用！
