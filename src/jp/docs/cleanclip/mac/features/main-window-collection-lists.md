# コレクションリスト - 手動管理

個別または複数の"アイテム"を選択し、右クリックメニューから"リストに追加"を選択すると、指定されたリストにアイテムをお気に入りに追加できます。

お気に入りリストはスマートリストによってフィルタリングでき、より多くの遊び方を実現できます。
