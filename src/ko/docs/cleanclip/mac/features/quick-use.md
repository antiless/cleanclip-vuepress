# 빠른 사용법

1. ⌘; 바로 가기 메뉴 호출
2. a-z를 입력하여 즉시 검색을 시작하고, 이미지를 검색하려면 image를 입력하고, 파일을 검색하려면 file을 입력합니다.
3. 스페이스바로 미리보기
4. ⇥/⇧⇥ 페이지 위아래로 이동
5. 숫자 키/엔터로 해당 항목 붙여넣기
