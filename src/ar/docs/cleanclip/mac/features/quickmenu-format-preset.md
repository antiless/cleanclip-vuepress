# تنسيقات الطباعة المسبقة الموحدة

بعد تعيين التنسيق مسبقًا، ستتم طباعة جميع العناصر وفقًا للتنسيق المحدد.

التنسيقات المدعومة حاليًا: اللصق كنص نقي، الحروف الصغيرة الكاملة، الحروف الكبيرة الكاملة، وغيرها.
