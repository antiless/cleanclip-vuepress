# 介绍

⌘' 打开粘贴队列后，复制的内容会自动进入队列，⌘V 会按序列依次粘贴队列里的下一项。

1. 点击底部的「上下箭头按钮」可反转列表
2.「拖拽项目」可调换顺序
3.「点击 #」 切换到指定项目
4.「粘贴后按下」选项会在粘贴后自动按下对应按键
