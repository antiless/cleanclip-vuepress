# تنسيق اللصق السريع

1. اضغط مع الاستمرار على رقم المحتوى لفتح قائمة التنسيق ثم اختر
2. اضغط مع الاستمرار على مفتاح المسافة لمعاينة النص المُنسَّق
3. اضغط قصارى جهدك على رقم التنسيق لتأكيد الاختيار
