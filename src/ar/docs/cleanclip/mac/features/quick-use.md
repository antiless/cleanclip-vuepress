# الاستخدام السريع

1. ⌘ ؛ استدعاء القائمة السريعة
2. a-z البحث المباشر ، إدخال image للبحث عن الصورة ، إدخال file للبحث عن الملف
3. مفتاح الفضاء المعاينة
4. ⇥ / ⇧⇥ الانتقال للأعلى / للأسفل
5. مفاتيح الأرقام / الإدخال الانتقالي للصق العنصر المقابل
