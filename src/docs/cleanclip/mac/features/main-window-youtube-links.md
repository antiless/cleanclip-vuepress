# 智能列表 - 根据条件智能筛选项目

利用智能列表，可以自由地设定筛选条件，从而获取到想要的内容。

目前支持的筛选条件有: 媒体类型、关键字匹配、正则匹配、字符长度、来源 App、来源设备、所在收藏夹。
