# 빠른 서식 지정 붙여넣기

1. 콘텐츠 색인 번호를 꾹 누르면 형식 메뉴가 나타납니다.
2. 공백 키를 길게 누르면 서식이 적용된 텍스트를 미리 볼 수 있습니다.
3. 형식 색인 번호를 짧게 눌러 선택을 확인합니다.
