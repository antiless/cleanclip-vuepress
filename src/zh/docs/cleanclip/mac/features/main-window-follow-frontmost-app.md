# 自动跟随前台应用

选择跟随前台应用模式后，应用筛选器会自动跟随前台应用变化，显示该应用的历史复制记录。

该模式与列表配合使用，可以扩展出很多玩法，可以为每一个 App 新增专用收藏空间，释放应用潜力。
