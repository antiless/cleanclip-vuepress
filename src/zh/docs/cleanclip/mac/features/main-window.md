# 介绍

主窗口是固定在屏幕上，拥有对剪贴板记录进行批量操作、管理、使用的窗口。


**筛选**：支持根据 App 筛选、筛选当前 App、 收藏列表筛选、智能列表根据多条件筛选。

**搜索**：在筛选范围内快速搜索。

**选择**：左键单击进行单选；**⌘+单击**添加项目；**⇧+单击**添加两次选择间的全部项目。

**使用**：双击粘贴单个项目，**拖拽**或右键菜单**批量操作**多个项目。
