# クイック使用法

1. ⌘キーを押すとクイックメニューが表示されます
2. a-zを入力すると直接検索が開始され、imageを入力すると画像を、fileを入力するとファイルを検索できます
3. スペースキーでプレビューが表示されます
4. ⇥/⇧⇥ キーでページを上下に移動できます
5. 数字キー/Enterキーで対応する項目を貼り付けることができます
